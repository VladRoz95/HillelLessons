  • git status - Проверяет репозиторий на наличие изменений.
  • git add . - Вносит изменения во все файлы репозитрия.
  • git add "name" - Вносит изменения в файл с именем "name".
  • git branch - Показывет в какой ветке мы находимся.
  • git checkout - Переключает между веткамию.
  • git checkout -b "name"- Создает новую ветку и переключает на неё.
  • git clone - Создает клон существующего репозитория.
